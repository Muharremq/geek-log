# GeekLog - Web Uygulaması

Bu proje, kullanıcıların izledikleri veya okudukları anime, manga, film, webtoon ve manhwa'ları takip etmelerine olanak tanıyan bir takip uygulamasıdır. React, TypeScript ve Bootstrap kullanılarak geliştirilmiştir. Durum takibi, puanlama ve etiketleme gibi özellikler içerir.

## Kurulum ve Çalıştırma

### Ön Gereksinimler

- Node.js (versiyon 14 veya üzeri)
- npm veya yarn

### Adımlar

1. **Proje bağımlılıklarını yükleyin:**

   ```bash
   npm install
2. Geliştirme sunucusunu başlatın:
   ```bash
    npm start
Artık uygulamanız http://localhost:4200/ adresinde çalışıyor olmalı!
